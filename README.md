# Alphabet Booklet (Kotlin / Android)

A simple Android app designed to help caretakers and teachers teach the alphabet to young children.

This project was developed as part of **CSC2002S — Computer Science 2002** at the **University of Cape Town**.

## About

The app presents an interactive alphabet booklet: a full A–Z overview screen, plus a dedicated page for each letter that a caretaker or teacher can use to walk a child through the alphabet one letter at a time.

## Features

- **Full alphabet overview** — a single layout displaying A through Z
- **Individual letter pages** — tap or navigate to a letter to view it on its own screen, useful for focused teaching
- **Simple, child-friendly interface** — minimal navigation suited to young learners and the adults guiding them

## Tech Stack

- **Language:** Kotlin
- **Platform:** Android

## Getting Started

### Prerequisites

- Android Studio (latest stable release recommended)
- Android SDK
- An Android emulator or physical device for testing

### Setup

1. Clone the repository
   ```bash
   git clone https://github.com/lramovha/alphabet_book_kotlin.git
   ```
2. Open the project in Android Studio.
3. Let Gradle sync and resolve dependencies.
4. Run the app on an emulator or connected device (`Shift + F10` or the Run button).
